Whole Brain Paragraph

My girlfriend was building a score board for her Derby club. I was asked to help when they could not get the hardware to work (she is a software person has not had to work with hardware in many years). I redesigned the PCB boards, wiring harnesses, and grounding scheme. While doing the electronics you have to use logic to solve the issues you also have to very creative. For example, doing the board layouts, you have to use parts in a way for the board to work, but you also want to lay it out in a way that some one that does not work with electronics can fix problem that might come up. Same with wiring harnesses. You want to run/build the cables so that if they need to be fixed and person with out any electrical experiences can do so.
